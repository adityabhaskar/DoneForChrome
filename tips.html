Extension Name

Log tasks from location bar / omnibox

Log your tasks without opening even popup. Simply, go to the location bar and type 'Done ' followed by your task. Providing a date in the yyyy-mm-dd format after 'Done' will add the done task to that specific date.

E.g. 'Done 2015-07-12 Finished final beta of Done extension #sideprojects' will add this task to my default iDoneThis team, on Sunday 12 July, with text 'Finished final beta of Done extension #sideprojects' and tag it as a #sideproject.

Keyboard shortcut

The default keyboard shortcut used by the extension is Ctrl + Shift + Z.. This brings up the popup to add a new Done. Just press the shortcut and start typing. Press 'Enter' to log the new Done, and get cracking on your list again.

You can change the keyboard shortcut to a different one here.

Add Goals

Start your task with a pair of square brackets - [ and ] - to log them as goals, instead of completed dones. The pair of [] symbolises an unchecked checkbox

Goals show up below your dones in your iDoneThis account, and can be checked off when completed.

Tagging

Including #tags in your tasks will tag them. You can see all tasks tagged with that hash tag in your iDoneThis account
